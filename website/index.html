Built for Modern Java Development

Zero-Config Developer Experience

Dev Services auto-provision databases, message brokers, and AI services as containers. One dependency, no boilerplate. A Dev UI for inspecting your running application, and a CLI that streamlines your entire development lifecycle.

Unified Observability

OpenTelemetry and Micrometer in a single dependency. Logs, metrics, and traces with OpenTelemetry and OpenInference semantic conventions out of the box. Auto-provisioned Grafana stack for local development.

Built-in Multitenancy

First-class multitenancy for enterprise SaaS applications. Flexible tenant resolution from HTTP headers and cookies, context propagation, tenant-aware caching, and built-in observability with MDC logging and OpenTelemetry integration.

Generative AI Ready

Integrated AI observability and evaluation platforms, document processing with Docling, and skills management to package, distribute, and install agent skills.

Automated Migrations

One-command upgrades for Spring Boot 4, Spring AI 2, and beyond. OpenRewrite recipes handle API changes, dependency bumps, and code refactoring automatically.
